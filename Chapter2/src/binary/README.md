# Binary, Type

진법의 사용
~~~java
int num = 10;
int bNum = 0B1010; // 10
int oNum = 012; // 10
int hNum = 0XA; // 10
===================================================
int num1 = 0B00000000000000000000000000000101; //5
int num2 = 0B11111111111111111111111111111011; //-5
~~~
숫자와 자료형
~~~java
=================== 정수 타입 ======================
# 데이터 타입     메모리크기        자릿수 
    byte  =     1바이트     (-128 ~ 127)
    short =     2바이트     (-32768 ~ 32767)
    int   =     4바이트     (-2147483648 ~ 2147483647)
    long  =     8바이트     (-263 ~ (263 - 1))
================== 부동 소수점 타입 ==================
    float =     4바이트     (소수 부분 6자리까지 오차없이 표현)
    double=     8바이트     (소수 부분 15자리까지 오차없이 표현)
==================== 문자형 타입 ======================
    char  =     2바이트
=================== 논리형 타입 =====================
    boolean=    1바이트     true or false
float 는 접미사로 F or f 를 사용하고, double 은 D or d 를 사용하지만 생략이 가능하다.

final 키워드가 변수,메서드,클래스 앞에 붙게 되면, 해당 엔티티는 변경할 수 없다.
